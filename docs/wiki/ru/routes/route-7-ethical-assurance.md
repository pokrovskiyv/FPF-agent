# Маршрут 7: Этический аудит (Ethical Assurance)

> Источник: `sections/routes/route-7-ethical-assurance.md`

## Краткое описание

Срабатывает на вопросы о скрытых предубеждениях, этических допущениях или конфликтах ценностей между командами на разных масштабах ("как проверить наличие предубеждений в нашем процессе?", "у инженеров и операций конфликт ценностей — как его показать?"). На выходе — реестр предубеждений, карта конфликтов по масштабам (человек / команда / организация / экосистема) и чек-лист этического аудита.

## Ключевые решения

- **Длина цепочки:** 5 секций при полной загрузке, 3 core.
- **Core-секции:** `D.2` (multi-scale ethics framework), `D.3` (holonic conflict topology), `D.5` (bias audit & ethical assurance).
- **Полная цепочка:** добавляет `D.1` (axiological neutrality) и `D.4` (trust-aware mediation calculus).

## Статус

Активен. Используется для задачи `ethical_assurance`. Шаблон резонера: "Карта конфликтов по масштабам → Реестр предубеждений (тип / место / риск / митигация) → Чек-лист аудита".
