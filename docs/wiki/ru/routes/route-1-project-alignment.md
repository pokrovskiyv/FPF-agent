# Маршрут 1: Зоны ответственности (Project Alignment)

> Источник: `sections/routes/route-1-project-alignment.md`

## Краткое описание

Срабатывает на формулировки вроде: "команды не понимают друг друга", "зоны ответственности размыты", "кто за что отвечает?", "передаём работу, но граница нечёткая". На выходе — карта "кто за что", потоки передачи работы между командами и перечень пробелов.

## Ключевые решения

- **Длина цепочки:** 8 секций при полной загрузке, 3 отмечены core для минимальной.
- **Core-секции:** `A.1.1` (семантическая рамка), `A.15` (согласование роль-метод-работа), `B.5.1` (explore → shape → evidence → operate).
- **Полная цепочка:** добавляет детали плана работ и слотового заполнения (`A.15.2`, `A.15.3`), гармонизацию методов (`F.11`), выравнивание между контекстами (`F.9`) и единый термин-лист (`F.17`).

## Статус

Активен. Используется для задачи `project_alignment`, детектируемой классификатором. Шаблон ответа (у резонера): "Зоны ответственности → Потоки работы между командами → Пробелы с ответственными".
